Three autoregressive approaches for generating bird vocalizations conditioned on species identity, trained on discrete audio tokens from the BirdSet XCM dataset.

Abstract

This project explores autoregressive generation of bird vocalizations conditioned on species identity. Three generation approaches are implemented and compared:

(1) LLaMA + SNAC — a custom causal language model trained from scratch on discrete SNAC audio tokens (32 kHz, 4-level codebook) with class-conditioned next-token prediction; (2) AudioGen + Species Conditioner — Meta's AudioGen-medium fine-tuned with a learned species embedding replacing the original text conditioner, trained via a three-stage freeze schedule with LoRA; (3) Ecogen VQ-VAE — a VQ-VAE operating on mel spectrograms, generating new samples via latent-space augmentation.

The training data comes from the BirdSet XCM dataset and optionally BirdCLEF competition data, with quality gating via a ConvNeXt classifier. Objective evaluation uses Inception Score (IS), Fréchet Audio Distance (FAD), and classification accuracy via BirdNET, ConvNeXt, and EnCodec embedders. Subjective evaluation is conducted through MOS listening tests delivered via Telegram.

Methods

From Scratch

LLaMA + SNAC

A 12-layer causal language model (768-dim, 12 heads) trained from scratch on interleaved SNAC tokens. SNAC encodes 32 kHz audio into 4 hierarchical codebook levels (4096 entries each), interleaved into 15 tokens per time step. Species conditioning is via a class token prepended before BOS.

Fine-tuned

AudioGen + Species Conditioner

AudioGen-medium (EnCodec-based) with the text conditioner replaced by a learned species embedding + linear projector. Three-stage training: conditioner only → LoRA (rank 16, alpha 32) + conditioner → refined LoRA + conditioner.

VQ-VAE

Ecogen

A hierarchical VQ-VAE operating on mel spectrograms. New samples are generated by augmenting latent codes via noise injection, interpolation between same-class latents, or sampling from per-class latent statistics.

Pipeline

1

Data Preparation

Download BirdSet XCM dataset, filter classes with ≥50 samples, chunk recordings into 10s segments, split into train/val/test. Optionally enrich with BirdCLEF data using ConvNeXt quality gating.

2

Tokenization

Encode audio into discrete tokens — SNAC (32 kHz, 4-level, 4096 entries) for the LLaMA backbone, or EnCodec (16 kHz) for AudioGen. Tokens are stored as compressed NumPy arrays.

3

Training

Train LLaMA from scratch with class-conditioned next-token prediction. Fine-tune AudioGen via three-stage schedule (conditioner → LoRA → refined LoRA). Train Ecogen VQ-VAE on mel spectrograms.

4

Generation & Reranking

Generate k candidate audio samples per species, score with BirdNET or ConvNeXt classifier, and select the best candidate. Supports temperature and top-k sampling.

5

Evaluation

Compute IS, FAD, and classification accuracy using BirdNET, ConvNeXt, and EnCodec embedders. Subjective MOS listening tests delivered via Telegram.
